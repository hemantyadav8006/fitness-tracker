---

## Getting Started

First, Clone the repo

```bash
git clone https://github.com/hemantyadav8006/fitness-tracker.git
```

---

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

---

#### Environment variables you need for this project

```docs
# MongoDB credentials
MONGODB_URI=

# JWT secret
JWT_SECRET=

# Bcrypt salt rounds
BCRYPT_SALT_ROUNDS=


# Email credentials
EMAIL_HOST=
EMAIL_PORT=
EMAIL_USER=
EMAIL_PASS=
EMAIL_FROM=
```
